pathSeparator :: Char #

The character that separates directories. In the case where more than one character is possible, pathSeparator is the 'ideal' one.

Windows: pathSeparator == '\\'
Posix:   pathSeparator ==  '/'
isPathSeparator pathSeparator

varPathSeparator :: Char Source #

Разделитель путей, записанных в переменной PATH.

(on Linux) varPathSeparator == ':'
(on Windows) varPathSeparator == ';'
