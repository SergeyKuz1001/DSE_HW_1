Methods

findFileByAbsPath :: AbsFilePath -> m (Maybe File) Source #

Поиск файла по абсолютному пути.

findFile :: (MonadFS m, MonadPwdReader m) => FilePath -> m (Maybe File) Source #

Поиск файла в текущей директории (если указан относительный путь) или в корневой директории (если указан абсолютный).

findFileAsExecutable :: (MonadFS m, MonadPwdReader m, MonadPathReader m) => FilePath -> m (Maybe File) Source #

Поиск файла, аналогичный findFile, но с использованием переменной PATH (через монаду MonadVarPathReader) если указано только имя файла.

doesFileExist :: (MonadFS m, MonadPwdReader m) => FilePath -> m (Maybe AbsFilePath) Source #

Проверка на то, что файл по этому пути (абсолютному или относительному) существует. Если файл существует, то возвращается абсолютный путь до него.

doesExecutableExist :: (MonadFS m, MonadPwdReader m, MonadPathReader m) => FilePath -> m (Maybe AbsFilePath) Source #

Аналогично doesFileExist, но при этом происходит поиск с использованием переменной PATH и проверка, что найденный файл может быть выполнен.

isReadable :: MonadFS m => AbsFilePath -> m Bool Source #

Проверка на то, что файл может быть прочитан. Возвращает False если файл не существует.

isWritable :: MonadFS m => AbsFilePath -> m Bool Source #

Проверка на то, что файл может быть перезаписан. Возвращает False если файл не существует.

isExecutable :: MonadFS m => AbsFilePath -> m Bool Source #

Проверка на то, что файл может быть выполнен. Возвращает False если файл не существует.
