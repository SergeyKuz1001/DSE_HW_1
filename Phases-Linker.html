linker :: MonadSelfReferenced m => Primitive -> m Primitive Source #

Функция для связывания команд, написанных через pipes. Работает только с последовательностями команд, остальные типы пользовательских запросов возвращаются без изменений.

Под "связыванием" команд здесь понимается определение типов потоков ввода и вывода для каждой команды, а также некоторая оптимизация плана исполнения пользовательского запроса.
